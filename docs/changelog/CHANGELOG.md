## [0.1.0] - 2026-02-13

### Added

- **Next.js 16 project setup** — App Router, React 19, TypeScript
- **shadcn/ui v4 integration** — Component library with Tailwind CSS v4
- **i18n with ~128 locales** — next-intl, namespace-based messages, 9 namespaces (common, navigation, hero, about, projects, skills, contact, footer, metadata)
- **Dark/light mode** — next-themes with system preference support
- **Motion animations** — Scroll and entrance animations via Motion library
- **Responsive layout** — Mobile-first design with breakpoints
- **Header, footer, navigation** — Layout components with locale-aware links
- **Locale switcher with search** — Searchable dropdown for ~128 locales, grouped by region
- **RTL support** — Automatic `dir="rtl"` for 9 RTL languages (Arabic, Persian, Hebrew, Kurdish, Pashto, Sindhi, Urdu, Uyghur, Yiddish)
- **Comprehensive documentation** — Architecture, i18n, guides, changelog
